A beautifully designed **yoga and wellness landing page** with a fullscreen video background, smooth navigation, and a courses section — built with pure HTML, CSS, and JavaScript.

---

## Features

- 🎬 **Video Background** — Immersive fullscreen looping background video
- 🧭 **Smooth Navigation** — Scroll-linked navbar with section anchors
- 📋 **Courses Section** — Display of available yoga courses
- ℹ️ **About Section** — Studio information and philosophy
- 🚀 **Get Started CTA** — Call-to-action for new members
- 📱 **Responsive Layout** — Adapts to all screen sizes

---

## Tech Stack

Pure HTML5, CSS3, and vanilla JavaScript — no frameworks or dependencies.

---

## Project Structure

```
Yogic_Wellness/
├── index.html           # Page structure and content
├── styles.css           # All styling and layout
├── script.js            # Navbar interaction and scroll behavior
├── logo.png             # Brand logo
├── bgvideo.mp4          # Background video file
├── thumbnail img/       # Course thumbnail images
└── Course video/        # Course preview content
```

---

## Getting Started

No installation required:

```bash
git clone https://github.com/pranotivarpe/Yogic_Wellness.git
cd Yogic_Wellness
open index.html   # or double-click the file
```

> Note: The background video (`bgvideo.mp4`) must be in the root directory for the hero section to display correctly.
